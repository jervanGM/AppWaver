Detailed Description

RTOS task and mutex management functions.

This header file provides the interface for creating, managing, and manipulating tasks and mutexes in a FreeRTOS environment. It defines various functions to create periodic and sporadic tasks, control task states, handle task notifications, and manage mutexes.
